사분면 알아내기
=======

문제 및 코드
-----
[문제](https://www.acmicpc.net/problem/14681)




# 제출 코드
'''
x=int(input())
y=int(input())
if x>0:
    if y>0:
        print('1')
    else:
        print('4')
else:
    if y>0:
        print('2')
    else:
        print('3')
'''
